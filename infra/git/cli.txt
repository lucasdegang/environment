# Tips:

## Basic commands:

git help
git remote update
git status
git branch [-l | -d | -a]
git checkout [branch]
git switch [branch]
git add [items | . ]
git commit [-a]
git fetch
git pull
git push
git push origin master
	The origin represents a remote name where the user wants to push the changes.
git.exe checkout [branch] --
git clone [url]
git fetch [-p | --prune]
	Update from server with local branches (new, deleted ...) Only git fetch cannot do this. 

## Create branch from master

git checkout master
git pull
git checkout -b [branch-name]
git add . 
git commit -m '[UPDATED] message'
git push --set-upstream origin [branch-name]

## Change commit message

## Squash commits on your branch

git checkout master
git pull
git checkout [your-branch]
git pull
git reset $(git merge-base master $(git branch --show-current))
git add -A
git commit -m '[SQUASH] message'
git push --force origin [your-branch]

## Rebase with master

git checkout master
git pull
git checkout [your-branch]
git pull
git rebase origin/master
git rebase [--continue | --skip | --abort | --edit-todo | --quit]
git push --force-with-lease origin [your-branch]
